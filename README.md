# HumanForYou-DataAnalysis
L’entreprise HumanForYou fait face à un fort taux de turn-over, mobilisant fortement son service RH pour recruter des profils équivalents et assurer la continuité de l’activité. Pour y remédier, elle sollicite des spécialistes afin d’analyser anonymement ses données et d’identifier les facteurs influençant le plus ce phénomène.
